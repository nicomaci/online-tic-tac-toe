# Online Tic-Tac-Toe

## Message Structure
```
[1, 0, 1, -1, -1, -1, 0, 1, 0]

            |
            v
        [X] [O] [X]
        [ ] [ ] [ ]
        [O] [X] [O]

1 = X
0 = O
-1 = Empty Square
```

Server and client will communicate back and forth these states until a victory condition is met.
